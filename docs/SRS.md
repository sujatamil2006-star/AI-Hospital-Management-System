# AI Hospital Management System

## Project Objective
Develop an AI-powered Hospital Management System that helps hospitals manage patients, doctors, appointments, billing, pharmacy, laboratories, and medical records efficiently.

## Users
- Admin
- Doctor
- Patient
- Receptionist

## Modules
- Authentication
- Patient Management
- Doctor Management
- Appointment Management
- Pharmacy
- Laboratory
- Billing
- AI Disease Prediction
- Dashboard

## Technologies
- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap
- Scikit-learn
